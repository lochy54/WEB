PROGETTO WEB
Il progetto si pone l’obiettivo di sviluppare l’applicazione web Social
Network for Music (SNM) che implementa un sito di gestione di
playlist musicali. SNM gestisce il processo di organizzazione di playlist
musicali di pi`u utenti e la loro condivisione. E composto da tre macro-scenari principali:

– gestione degli utenti;

– gestione delle playlist;

– gestione delle condivisioni;

L’elenco delle canzoni,artisti e generi
viene acquisito tramite le API REST del portale Spotify (https://developer.spotify.com/documentation/web-api/reference).
